![Tests status](https://img.shields.io/github/workflow/status/issy/dvla-cli/test?label=tests)

A set of commandline utilities to interact with DVLA vehicle APIs.

## Features

- Fetch MOT tests
- Fetch vehicle information

## Usage Examples

`dvla tests RJ06JOU`

`dvla vehicle RJ06JOU`
